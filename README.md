# Swarm Demo — AI Model Price Comparison

A web app to compare AI model pricing across providers (OpenAI, Anthropic, Google, Meta, Mistral, etc).

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Shadcn/ui components

## Features (target)
- Sortable/filterable table of AI models
- Price per 1M tokens (input/output)
- Context window, modality, provider info
- Data sourced from OpenRouter API
- Dark mode
- Mobile responsive
